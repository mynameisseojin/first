## LLM 모델 비교 선정 보고서 
비교축 :
정확성: 입력된 정보와 요청 사항을 정확하게 반영하는 정도
형식 준수: 이메일 구조(제목, 인사, 본문, 마무리, 서명)를 잘 지키는지
정중함 및 자연스러움: 교수에게 적절한 표현과 자연스러운 문장 구성
문맥 이해: 결석 사유 및 요청 의도를 정확히 파악했는지
실용성: 실제로 바로 사용 가능한 완성도

사용환경 : 모두 웹으로 진행하였다.

GPT

형식 준수: 4점 (제목 없음, 구조는 안정적이나 완전한 형식은 아님)
정중함/자연스러움: 5점 (매우 자연스럽고 과하지 않은 정중함)
문맥 이해: 5점 (출석 인정 + 증빙 문의를 명확히 처리)
실용성: 5점 (즉시 사용 가능한 수준)

정중함/자연스러움: 5점 (자연스럽고 부드러운 표현)

최종선정결론

이 과업은 교수님께 보내는 공식 이메일 작성으로, 형식의 완성도와 정중한 표현, 그리고 불필요한 내용 없이 핵심을 전달하는 능력이 중요한 기준이다.
비교 결과, Claude 모델은 제목을 포함한 이메일 구조를 가장 명확하게 준수하였으며, 인사, 본문, 마무리 구성이 구분되어 있어 형식적 완성도가 가장 높았다. 또한 불필요한 추가 설명 없이 이메일 초안 자체에 집중하여 결과를 제시한 점에서 실용성이 높다고 판단된다.
반면 GPT는 자연스러운 표현에서는 우수했으나 제목이 누락되었고, Gemini는 완성도는 높지만 추가적인 안내 문장이 포함되어 있었다.
따라서 이 과업에서는 형식 준수와 구조적 완성도가 가장 우수한 Claude 모델을 최종 모델로 선정하였다.
